# -Voyage-JavaScript-
《远行》是一个关于控制、观察与反转的视听交互装置。 寂静的观察： 无摄像头时，装置处于寂静世界。 有摄像头时，发光的星体在虚空中孤独游弋，背景粒子随洋流流动，线条作为信号从边缘向内延展。这是一种不被观测时的量子叠加态，宁静深远。 引力的介入： 伸出手，星体立刻感知到引力，迅速转向并吸附和受牵引于指尖。伴随着移动速度的快慢，生成式音频会发出类似风掠过虚空的呼啸声。 负片世界： 握紧拳头，原本黑暗的宇宙瞬间反相，背景白化，光影黑化，音乐从清脆的风铃突变为压抑的低频脉冲。投射极端的控制能量的坍缩与维度的翻转。 粒子与回响： 背景中的微小粒子具有物理属性，会被星体的力场推开。双重声景，平静时的和弦旋律 &amp; 负片时的锯齿波低音，可随着交互自由切换。
